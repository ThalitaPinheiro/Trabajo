Este trabajo consiste en una especie de agenda de compromisos.

Bien sencillo, fue desarrollado con java, utilizando el Framework JSF y primefaces.
He utilizado el glassfish y probado la aplicación en Google Chrome.


En "hibernate.cfg" están las opciones de configuración de la base de dados. 
He añadido el modelo de la base de datos (hecha en el Workbench) y el script de creación de las tablas en una carpeta que llame "Bases de Datos".


Las librarias , caso sea necesario, estan duplicadas en el proyecto. He ponido ellas en una carpeta con el nombre de "lib", dentro de la carpeta principal del proyecto. 
He hecho eso porque he tenido problemas de reconocimiento de las libs cuando he bajado el proyecto en otro ordenador. 
Entonces, creo que sea mejor asi, para que sea mas sencillo a localización de las librarias.
